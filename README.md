# Gerenciador de reuniões
Gerenciador de reuniões feito na disciplina de Engenharia de Software e Sistemas do curso de Ciência da Computação na Universidade Federal de Pernambuco. Esse gerenciador é feito utilizando Angular 13 e está conectado a um servidor, o que permite criar reuniões e postar recados em um mural. Para fazer funcionar, é preciso ter o Angular instalado e executar o comando "npm start" tanto na pasta "mtm-gui", quanto na "mtm-server".
